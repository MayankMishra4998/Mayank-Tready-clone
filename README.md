# Mayank‑Trendy‑Clone

## 📌 Overview
Mayank‑Trendy‑Clone is a modern, responsive portfolio website designed to showcase professional skills, services, and creative projects. It features clean navigation, polished UI, and sections like **Home, About, Services, Portfolio, and Clients** — making it ideal for developers, designers, or freelancers to present their work.

## ✨ Features
- **Landing Page**: Sleek introduction with branding and navigation  
- **About Section**: Highlights background, expertise, and vision  
- **Services**: Prototype design, Android/iOS development, and web app solutions  
- **Portfolio**: Showcase of selected projects with visual emphasis  
- **Clients**: Section to highlight collaborations and professional relationships  
- **Responsive Design**: Optimized for desktop, tablet, and mobile  

## 🛠 Tech Stack
- **HTML5, CSS3, JavaScript** for structure and interactivity  
- **Responsive Layouts** for cross‑device compatibility  
- **Netlify Deployment** for fast and reliable hosting
